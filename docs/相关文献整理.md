# 相关文献整理

## 一、为什么这里单独整理文献

本项目的论文不能只写新能源汽车背景和系统开发流程，还需要有与“智能预测”和“电池寿命建模”相关的研究支撑。因此，这里单独整理一组更贴合本项目的深度学习相关文献，便于直接写入绪论和相关工作章节。

## 二、建议保留的两类文献方向

为了和本项目保持一致，建议重点保留两类文献：

- 电动汽车真实工况能耗预测
- 锂电池 SOH / RUL 预测

这样能分别对应项目中的两条研究线索：

- 行车日志驱动的业务预测与分析
- NASA 电池寿命实验

## 四、绪论中可以怎样概括

### 1. 能耗预测方向

可以概括为：

近年来，围绕电动汽车真实工况能耗预测的研究逐步从传统经验模型向数据驱动方法演进。Pan 等人的研究表明，结合短行程分段与深度学习的方法能够提升对复杂驾驶工况的适应能力。Ko 等人进一步说明，基于 GPS 轨迹数据的深度学习模型可以支持车辆能耗监测与预测。Pamuła 等人的研究则从电动公交场景验证了深度学习方法在能耗预测中的实际价值。总体来看，深度学习已成为电动车能耗预测的重要研究方向，但模型效果仍与数据质量、特征组织方式和应用场景高度相关。

### 2. SOH / RUL 方向

可以概括为：

在锂离子电池健康状态估计与剩余寿命预测方面，研究重点集中在对退化时序特征的有效建模。Li 等人提出了面向电动汽车锂电池的端到端神经网络框架，可同时完成 SOH 估计和 RUL 预测。Zhu 等人采用注意力机制与 CNN-BiLSTM 结合的方式增强了对局部退化特征和长时依赖的联合刻画。Zhou 等人则验证了时序卷积网络在长序列寿命预测中的潜力。Ren 等人和 Wang 等人的研究进一步表明，卷积与循环网络组合建模以及迁移学习方法，都能够为电池寿命预测带来新的改进思路。

## 五、这些文献与本项目的关系

这些文献不能只当作“为了凑参考文献数量”而存在，它们在本项目里分别有清晰的落点：

- [16]-[18] 用于支撑行车日志驱动的能耗分析和智能预测背景。
- [19]-[23] 用于支撑 NASA 电池 SOH / RUL 实验的研究合理性。

论文里最好明确写出：

“本文并不追求完整复现上述文献中的复杂模型，而是在综合已有研究思路的基础上，结合当前数据条件与毕业设计交付要求，完成一套以业务系统为主、以实验验证为补充的项目化实现。”

这样既显得论证充分，也不会给自己挖下“为什么没有完全复现文献模型”的坑。
