# 相关文献整理

## 一、与项目直接相关的文献方向

本项目的相关文献主要分为两类：

- 电动汽车真实工况能耗预测
- 锂电池 SOH / RUL 预测

这两类方向分别对应项目中的业务预测链路和 NASA 电池寿命实验链路。

## 四、文献使用边界

这些文献用于支撑项目中的预测方法背景和电池寿命实验背景，并不意味着项目完全复现了文献中的全部复杂模型。当前实现重点是结合现有数据条件完成项目化落地。
